# Loom — Brand System

A self-initiated brand sprint. One week to build a complete brand system 
for a concept product: Loom, a textile research studio for archivists, 
curators, and makers.

This repo contains the landing page code. The full brand system 
(foundations, tokens, components) is documented in the case study.

## Case study

[karendettmar.com/projects/loom](https://karendettmar.com/projects/loom)

## What's in this repo

- Landing page built from the Loom design system
- Design tokens implemented as CSS custom properties
- Component library: buttons, text inputs, cards
- Responsive layout

## Tech stack

- Astro
- TypeScript
- CSS custom properties (design tokens)
- Figma (source of truth for brand system)

## The brief

Build a brand system for a textile research studio from scratch. 
Constraints: one week, concept product only, no client. Goals: show how 
brand decisions translate into design tokens and components, demonstrate 
archival and craft-forward visual direction.

## Local development

```bash
npm install
npm run dev
```

This is a concept project, not a deployed product. The landing page
